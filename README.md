# Gooey Search Transcriber

## 📌 Overview
A web application that transforms voice search results into structured and readable transcripts, facilitating data consumption and analysis.

---
## 🚀 Features
- Automatic transcription of search results
- Text processing and formatting
- Clean and simple user interface
---

## 🛠 Technologies used
- JavaScript
- Whisper API
- CSS3
- FFmpeg
- Text processing logic

---
## 📦 Installation
```bash
git clone https://github.com/Zagarsix/Gooey-search-transcriber
cd Gooey-search-transcriber
```

---
## 🚀 Running the Application Locally
This project is divided into two parts:

- backend/
- frontend/

Both services must be running simultaneously.

### 1️⃣ Start the Backend
Open a terminal and run:
```bash
cd backend
npm install
npm run dev
```
### 2️⃣ Start the Frontend
Open a second terminal and run:
```bash
cd frontend
npm install
npm run dev
```

### 🌐 Access the Application
Once both servers are running:

Frontend: http://localhost:3000
Backend: http://localhost:5000 (or configured port)

---
## ⚠️ Notes
- Make sure both services are running at the same time
- Check environment variables if the frontend cannot connect to the backend
